# UQ in Real-Time System

This repository contains Jupyter notebooks for analyzing Autonomous Arena(https://www.kaggle.com/datasets/firstofhisname/indoor-car-track) and Traffic Signs Detection(https://www.kaggle.com/datasets/pkdarabi/cardetection).

## Project Description

The analysis provides insights into Uncertainty Quantification in real-time systems. Each notebook processes the corresponding dataset and generates relevant visualizations and outputs.

## Requirements

To run these notebooks, you need the following:
- Python 3.8 or above
- Jupyter Notebook or JupyterLab
- Required Python libraries: pandas, numpy, matplotlib, etc. (see below for setup instructions)
